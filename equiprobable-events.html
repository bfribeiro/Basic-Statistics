Basic Statistics With R

3.1 Equiprobable Events

Equiprobable events are those in which all elements of the set have the same probability of occuring. In other words, the events are equiprobable if \(P(e_i)=\frac{1}{n}\) where \(n\) is the number of events of the sample set.

A simple example of equiprobable events is a die. Every number has exactly the same probability of occuring.

# Single roll of a die can be expressed by:

sample(1:6,1) # Sample function generates draws, at random, a sample for a chosen set
## [1] 2

# Everytime you run the function, there will be a different value.

Probability does not tell that an event will certain happen, it just tells the likelihood of that event happen. As you repeat the experiment, and take note of results, you see that the events tend to have the same probability.

As you can see, as the number of repetitions increase the frequency in which each number occurs tends to be the same value.
