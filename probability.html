Basic Statistics With R

Chapter 3 Probability

To keep it simple: probability measures how likely something is to happen. As said before, if you toss a coin there are two possible outcomes: heads or tails. If you throw a single die, there are six possible outcomes: 1, 2, 3, 4, 5 or 6. In general, the probability function is given by:

\(Probability\ of\ an\ event= \frac{Number\ of\ favourable\ events}{Total\ possible\ outcomes}\) or \(P(A_i)= \frac{A}{S}\)

To illustrate, if toss two coins, how likely is to get two heads?

As we did in the previous chapter, we, first, need to build the sample space:

coin_1<-c('heads','tails')
coin_2<-c('heads','tails')

expand.grid("Coin 1"=coin_1, "Coin 2"=coin_2)
##   Coin 1 Coin 2
## 1  heads  heads
## 2  tails  heads
## 3  heads  tails
## 4  tails  tails

So we have 4 possible outcomes:

\(S=\{(heads,\ heads),\ (tails,\ heads),\ (heads, \ tails),\ (tails,\ tails)\}\).

That said, to calculate the probability of getting two heads we get the number of events in which two heads happen and divide by the total number of events: \(P(heads, heads)=\frac{1}{4}\).

# Defining Our sample space
S<-expand.grid("Coin 1"=coin_1, "Coin 2"=coin_2)

# Defining the number of possible outcomes:
A<-subset(S, grepl("heads", S$`Coin 1`) & grepl("heads", S$`Coin 2`))
S
##   Coin 1 Coin 2
## 1  heads  heads
## 2  tails  heads
## 3  heads  tails
## 4  tails  tails
A
##   Coin 1 Coin 2
## 1  heads  heads
Prob_A<-1/4

# In proportions:
Prob_A
## [1] 0.25

# As percentage

Prob_A*100
## [1] 25

The same principle is applied to 3, 4, 5, n coins or a die, two dice, three dice, etc.

You’ve probably noted that we did exaclty the same thing we did in the previous chapter: we counted the sets. Actually, the same principles that were applied to set theory can be applied in probability, even the operations. That happens because the number of favourable events we look for, is a subset of a larger set, the sample space: \(A \subset S\).

Keep in mind that:

  • The probability of an event \(A\) is often written as \(P(A)\)
  • \(P(S)=1\)
  • If \(A\) and \(B\) are disjoint, then \(P(A \cup B) = P(A) + P(B)\) (exactly like set theory)
  • The probability of an event will always be between 0 and 1: \(\{0 \leq P(A) \leq 1|A \subset S\}\)
  • \(\cup\) means addition and \(\cap\) means multiplication

From this point, I’ll write some “technical” aspects of probability. Nevertheless, are just for clarity and you can consider it a cheatlist for solving problems later on.

Some theorems that apply to probability are:

  1. The sum of the probabilities of all sets is equal 1: \(\sum_{i=1}^{n}P(A_i)=1\)
  2. If an event is impossible, then \(P(\Phi)=0\)
  3. For all event in which \(A \subset S\), temos que \(P(A)+P(A^{c})=1\)
  4. If events \(A\) and \(B\) are subsets of \(S\), then \(P(A \cup B)=P(A)+P(B)-P(A \cap B)\). However, if \(P(A \cap B)=\Phi\) then \(P(A \cup B)=P(A)+P(B)\).
  5. If events \(A\) and \(B\) belongs to the same sample space \(S\), then \(P(A \cup B) \leq P(A)+P(B)\)

Some practical applications

  1. Given \(P(A)=x, P(B)=y\) and \(P(A \cap B)=z\), solve:
  1. \(P(A^{c} \cup B^{c})\)
  2. \(P(A^{c} \cap B^{c})\)
  3. \(P(A^{c} \cap B)\)
  4. \(P(A^{c} \cup B)\)

Solutions:

  1. \(P(A^{c} \cup B^{c})=P(A \cap B)^{c} \therefore 1-P(A \cap B) \therefore 1-z\)
  2. \(P(A^{c} \cap B^{c})=P(A \cup B)^{c} \therefore 1-P(A \cup B) \therefore 1-\{P(A)+P(B)-P(A \cap B)\}\)

Since we already have \(P(A), P(B)\) and \(P(A \cap B)\): \(1-x-y+z\)

  1. \(P(A^{c} \cap B)=P(B-A) \therefore P(B)-P(A \cap B) \therefore y-z\)
  2. \(P(A^{c} \cup B)=P(A^{c})+P(B)-P(A^{c} \cap B) \therefore (1-x)+y-(y-z) \therefore 1-x+z\)

In the previous chapter there was a topic about the Principle of Inclusio-Exclusion - PIE. Hence probability follows the “same properties” of sets theory: \(P(A \cup B \cup C)=P(A)+P(B)+P(C)-P(A \cap B)-P(A \cap C)-P(B \cap C)+P(A \cap B \cap C)\)

The formulas above will help to solve problems later.
